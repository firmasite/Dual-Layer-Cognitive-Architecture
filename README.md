## üöÄ How to Use (30 Seconds Setup)

You don't need to be a coder. You just need to **Copy & Paste**.

### Step 1: Get the Code
1.  Open the file [`instructions.txt`](instructions.txt) in this repository.
2.  **Copy** the entire text.

### Step 2: Install into your AI

#### üü¢ For ChatGPT (Plus/Team/Enterprise)
1.  Click your profile name in the bottom-left corner.
2.  Select **"Customize ChatGPT"**.
3.  Look for the box that says: *"How would you like ChatGPT to respond?"* (The bottom box).
4.  **Paste** the text there.
5.  Make sure "Enable for new chats" is ticked and hit **Save**.

#### üîµ For Google Gemini (Gemini Advanced / Gems)
1.  Go to Gemini and click on **Gem Manager** (or "New Gem").
2.  Name your Gem (e.g., "Super Squad" or "Deep Thinker").
3.  In the **Instructions** box, **Paste** the text you copied.
4.  Click **Create**.
5.  *Tip: Whenever you have a hard task, select this Gem instead of the standard chat.*
